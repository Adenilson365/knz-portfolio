Infraestrutura como código AWS

Provisionei ambiente kubernetes multi-cluster na AWS usando terraform com pipeline Github Actions. Usei gitflow para versionamento e controle de alterações com segmentação por environments, proteção de branchs e approve required. Terraform usei remote state com S3 e DynamoDB para lock além controle de contexto para deploy em diferentes ambientes. Aws organizations para separação de ambientes em diferentes accounts.
